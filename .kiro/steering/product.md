# IPO Tracking Mobile Application

## Product Overview

This is a React Native/Expo mobile application for tracking Initial Public Offerings (IPOs) in the Indian stock market. The app provides real-time IPO data, market indices, stock information, and allotment checking functionality.

## Key Features

- **IPO Tracking**: View active, upcoming, closed, and listed IPOs with detailed information
- **Grey Market Premium (GMP)**: Real-time GMP data for better investment decisions
- **Market Indices**: Live Nifty and Sensex data with 30-second auto-refresh
- **Stock Data**: Most traded stocks, gainers/losers by market cap categories
- **Allotment Checker**: Check IPO allotment status using PAN number
- **Multi-platform**: Supports iOS, Android, and Web platforms

## Target Users

- Individual investors interested in IPO investments
- Stock market enthusiasts tracking market trends
- Users seeking IPO allotment status updates

## Business Context

The app focuses on the Indian stock market ecosystem, providing comprehensive IPO-related information to help users make informed investment decisions. It emphasizes real-time data accuracy and user-friendly mobile experience.
